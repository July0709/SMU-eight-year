# 临床思维思维训练

## 临床医生思维方法
### 快速思维 （非分析型思维）
病例库的储存
### 慢速思维（分析性思维）
通常从解剖-病理-生理角度解释相关症状和体征，推理得出解决方案
eg,.止血带扎在中1/3，桡神经走行在浅表；不扎在前臂，因为前臂骨间膜中有丰富的侧枝循环

汇报病情（逻辑思维能力）--必要信息--问诊--鉴别诊断

#### 例2. 鉴别诊断能力

##### 急诊病人接诊查因
青年男性，突发晕厥。求晕厥可能的原因？
1. 神经源性：中暑--血管迷走性相关
2. 内分泌：低血糖--吃早饭了吗
3. 心源性：心脏病--心悸胸痛--晕厥前有无其他症状
4. 神经源性：癫痫发作--倒地有无大小便失禁，不自主运动，有无其它症状
5. 创伤性：撞击--颈动脉窦--有无外伤冲撞

青年男性，晕厥伴呼吸苦难，端坐呼吸。
1. 心律失常？主动脉狭窄？
2. 肥厚性心肌病？肺栓塞？

--以前就有吗？多久了？多没有发展变化？
答：近一年自觉运动耐力下降？

之前就诊经历？用药史？
答：无心悸，有胸痛。

是否能排除肺栓塞？
答：劳力性心绞痛？肺栓塞？

长途旅行？长期卧床？有没有查出血栓肿瘤？（无）

主动脉狭窄？肥厚梗阻性心肌病？

以前身体好吗？有没有得过什么疾病？--心瓣膜病？主狭？

--查体？（胸骨右缘第二肋间4/6期喷射样杂音，杂音没有变化是主狭，有变化是梗阻性心肌病）

--辅助检查？（超声心动图）

##### 总结
青年男性，因突发短暂自限性晕厥伴进行性呼吸困难，阵发性胸痛收入我院。
