# 慢性心力衰竭
## 瓣膜病
梨型心--二狭
心尖部舒张期隆隆样杂音--二狭
胸骨右缘第二肋间收缩期喷射样杂音--主狭
胸骨左缘第二肋间机器样连续性杂音--动脉导管未闭

## 心力衰竭分级与Killip分级
### Killip分级 
+ 首先看有无心源性休克，有心源性休克就是Ⅳ级
中下肺布满湿啰音 ＞50%
### 心衰NYHA分级
+ 6分钟步行实验（150 450）

## 心衰的临床表现
+ 左心衰以肺循环淤血与心排血量降低为主要表现
  + 出现最早：劳力性呼吸困难
  + 最严重症状：心源性哮喘
  + 心脏杂音：二尖瓣关闭不全
  + 脉搏：交替脉
+ 右心衰竭以体循环淤血为主要表现
  + 最常见症状：消化道症状
  + 主要体征：颈静脉征
  + 特征性：肝颈静脉反流征阳性
  + 心脏杂音：三尖瓣关闭不全杂音、
  + 脉搏：奇脉（PS水冲脉--甲状腺功能亢进、动脉导管未闭、严重贫血）
+ 警惕：左心衰肺淤血症状缓解，警惕全心衰
