# 高血压

## 血压分类和定义
舒张压分级以20为一个单位进行分级，收缩压以10为一个单位进行分级
（正常高值，关口前移）
原发性占90%，继发性10%

## 病因
遗传因素，有明显的家族聚集性
环境因素：
饮食：高钠、饮酒
叶酸缺乏（同型半胱氨酸血症）--补充叶酸
脑力活动，精神紧张

## 发病机制--二级以上联合用药
肾素-血管紧张素系统RAS--ACEI,ARB
钠与高血压--利尿剂--氢氯噻嗪
神经机制（交感神经活性亢进）--年轻高血压多见--β受体阻滞剂--洛尔、洛克
血管内皮功能异常--内皮素、NO--肺动脉高压的时候有针对NO的药物
胰岛素抵抗

## 病理生理和病理
老年高血压病人而言，单纯收缩期高血压是最常见的类型
靶器官损害（心、脑、肾、视网膜）

## 症状
常见表现：
**头晕、头痛**、疲劳、**颈项板紧**、疲劳、心悸、视力模糊、**鼻出血**
典型高血压头痛：血压下降后缓解

## 临床思路
原发还是继发？有没有并发症？
有没有高血压？既往有高血压但服药，也算高血压

## 预后重要因素（未来五年、十年发生心血管风险都是不一样的）
**1234，2234，3444**
注意糖尿病的区分有无临床合并症，有并发症的都是很高危
血压水平、心血管危险因素、靶器官损害、伴随临床疾病

## 治疗（什么时候开始用药、控到什么时候）
最终目的：减少高血压患者心、脑血管病的发生率和死亡率
+ 目标值：
    + 普通病人：140/90mmHg
    + 糖尿病、慢性肾脏病、心力衰竭或稳定性冠心病患者<130/80mmHg
    + 老年收缩期高血压<150mmHg（能耐受要到140：没有出现头晕、体位性低血压）
+ 血压控制策略
  + 尽早降压、但不宜过快
  + 老年人要降压慢一点，规避脑梗
+ 降压药物选择（ABCD）
  + 长效联合使用，多途径用药
    + 降压药什么时候开始用
    + 一级优先改善生活方式
    + 二级/高血压合并糖尿病患者
  + 利尿剂（糖尿病、痛风、低钾高钾）
    + 常：呋塞米（排钾）和螺内酯（保钾）联用
  + β受体阻滞剂（禁忌症：病态窦房结抑制药）
    + 美托洛尔选择性β1受体拮抗剂
    + 普萘洛尔选择性（甲亢病人才用）
    + 阿罗洛尔（α和β连用）
      + **α受体降压效果好，容易体位性低血压，建议夜间使用**
  + 钙离子升高（会导致血管收缩）（副作用：面色潮红，常与β受体连用降低副作用）
    + 起效迅速，要做成缓释片（eg.,尼群地平）
    + 降压迅速，对血脂和血糖没有明显影响
    + 二氢吡啶（地平）
    + 非二氢吡啶（一般心律失常）
  + ACEI（副作用：缓激肽保留的多，干咳--多了换沙坦）
    + 普利
    + 糖尿病适合（可以改善胰岛素抵抗）
    + 可以减少尿蛋白（侯凡凡院士）
  + ARB
    + 起效缓慢，持久而平稳
  + 联合用药方法
    + 降压药使用原则
      + 合并有心力衰竭者适宜选择ARNI/ACEI、利尿剂
      + 老年人收缩期高血压者，宜选择利尿剂、长效二氢吡啶钙通道阻滞剂
      + 合并糖尿病、蛋白尿或轻、中度肾功能不全者（非肾血管性），可用ACEI
      + 合并冠心病的患者，优选β受体阻滞剂或ACEI（尤其伴收缩功能不全者）
      + 伴有脂质代谢异常的可选用α1受体阻滞剂，不宜选用β受体阻滞剂及利尿剂
      + 伴妊娠者，不宜用ACEI、血管紧张素II受体阻滞剂
      + 对合并支气管哮喘、抑郁症、糖尿病患者不宜用β受体阻滞剂，通风患者不宜用利尿剂、心脏起搏传到障碍者不宜用β受体阻滞剂及二氢吡啶类钙通道阻滞剂
+ 介入治疗
  + 肾动脉去神经术--难治性高血压--大部分人做完手术之后还要进行服药，只是药含量较少

## 特殊类型高血压
**难治性高血压** 三种以上包或4种药物才能使药物达标--多为找不到原因的
重点：
1. 高血压危象（舒张压＞150mmHg）
2. 高血压急症（严重升高伴有靶器官损害--心、肾、眼底，血压严重升高BP＞180/120mmHg）
3. 高血压亚急症（血压严重升高但没有靶器官损害）
4. 评估：MRI看有无头颅病变
   血压升高的程度不是区别高血压急症和亚急症的标准
5. 治疗原则（执业医会考）
   1. 一定要静脉快速降压（静脉联合口服双管齐下）
      1. 控制性降压（时间线记忆）P271（重点）
